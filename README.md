# 低能耗家用电器
----

使用机器学习的办法,试图寻找耗电量与房屋周围的温度、湿度等等之间的关系.
# 必备条件
请提前下载好Python、numpy、pandas、matplotlib和**skleran**开发环境,否则该项目无法使用,
我的开发环境是直接安装anaconda3.4,点击此处下载(https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/)
